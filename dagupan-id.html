The Core Problem

Fragmented paper records and siloed databases often lead to identity theft, redundant registrations, and slow service delivery. The Dagupan One ID provides a single source of truth for all resident interactions with the local government.

Technical Infrastructure

The system is architected to handle city-wide traffic with high availability. By leveraging a robust PHP/Laravel framework, it ensures sub-second response times.

  • Scanner API: Secure endpoints for hardware integration in public offices.
  • AES Encryption: Built with high-level data encryption to remain compliant with privacy laws.
  • Biometric Ready: Scalable architecture that supports future biometric modules.
  • Role-Based Security: Granular permissions for different LGU departments.

Specifications

  • Laravel Framework
  • MySQL Relational DB
  • RSA/AES Data Encryption
  • Real-time Sync API
